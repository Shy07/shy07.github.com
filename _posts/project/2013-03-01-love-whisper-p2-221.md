## 2  
### The Prologue  
　　2016年4月4日  
　　天空灰蒙蒙的，让人看不到一丝生气，更无法让人联想起现在已是万物苏生的初春。  
　　清明时节雨纷纷吗？陈欣儿低下仰视天空的头，视线回到眼前的高速公路上。挤出拥堵的扫墓车流，陈欣儿驾车驶出了A9高速收费站，左拐后一口气冲上了通往静园公墓的高架。  
　　花了不到五分钟，陈欣儿就已经停好车，拿着在市区自家门口摘下的玉兰花枝走向公墓的最深处。  
　　“这是你最喜欢的玉兰哦。”陈欣儿微笑着放下花枝，忽然注意到右边的墓碑，“抱歉啊，伯母，忘了给你带礼物。”陈欣儿调皮的吐了吐舌头，缓缓直起身子。  
　　“梦里绯红花千片，珍珠露下朱唇努。”陈欣儿轻轻吟诵着墓志铭上的诗句。  
　　这是凌仲文学生时代作的诗，题为“梦中见雨淋玉兰花落有感聊为百余字以记”。在仲文的博客网站上，仲文是这样解释的：诗中的玉兰与其说是具象的花，不如说是陈欣儿在他心中的记忆。陈欣儿高贵红艳的形象在他心中完美地和红玉兰重合了起来。他说这个梦是他这一生最不愿意见到的一幕，但也是最甜美幸福的一幕。因为那个自己深爱的女孩为自己哭了。  
　　你已经预见到这一幕了吗？凌仲文——注视着新立墓碑上的笑脸，陈欣儿再也无法维持住笑容，本已干涸了的眼泪再次奔涌而出。好狡猾啊，凌仲文，自己抛下一切，微笑着看着别人为你淋雨为你哭泣。嘴上说着最不愿意见到，但是心里却偷偷窃喜是吗？凌仲文！  
　　大约两周前，陈欣儿从一个陌生而熟悉的号码那里收到一条信息：你好，陈欣儿。一直匿名发给你生日祝福，真是抱歉啊。我是凌仲文，永别了。——时间，赋予我们成长。而“记得”，就是对彼此最珍贵的馈赠。Best wishes.  
　　一头雾水的陈欣儿马上又接到了一个电话，电话的另一头称自己是仲文的表哥，在收拾仲文遗物的时候，发现了这条存在移动终端服务器里的信息草稿。“冒昧地把它发送给你真的很对不起，但这应该是他能给你留下的最后回忆了。他的葬礼希望你也能来参加，就在朱家角的静园公墓。具体事宜在他的博客网站上可以查到，网址是Shy07.com。”  
　　而仲文的博客网站中给陈欣儿留下印象最深的便是这首题为“梦中见雨淋玉兰花落有感聊为百余字以记”的诗。自己在他的印象中竟是这样的遥不可及吗？陈欣儿的身体一阵颤抖，是肉麻恶心还是幸福眩晕？连陈欣儿自己都不知道。但是一想到有个生命就这样消逝了，陈欣儿的泪水便无法遏制地流了下来。  
　　“圣母之泪一样的泪水。”  
　　“诶？”仿佛是幻听一样，陈欣儿耳边响起了凌仲文的声音。  
　　“圣母之泪，那是法利特尔世界最美的宝石。”  
　　陈欣儿察觉到声音来自自己的身后，猛地转过身去，便失声惊叫了起来，“凌仲文？”  
　　“不，我不是仲文。”一名长相和凌仲文几乎完全一样的男子，持着花束缓缓向她身后走去，优雅地把花束放在凌仲文墓碑旁边的墓碑上，轻轻对着凌仲文母亲的墓碑喊了一声“妈妈”。然后，男子直起身，毫无做作地伸出右手，“你好，我叫凌伯言。”  
　　“伯——”陈欣儿轻轻地和凌伯言握了握手，“是仲文的哥哥？”  
　　“只是早出生几分钟而已，”伯言微微一笑，像要把气氛缓和下来，“而且母亲也没把我俩分长幼吧。你看，虽然伯字排行在仲字前面，但是母亲最喜欢的文言里，言字可在文字之后哦。”  
　　陈欣儿顺着伯言的手指看向墓志铭，“为什么从来没听过啊，凌仲文还会有一个哥哥？你们是同岁的吧，应该一起上学的吧？我怎么——”  
　　虽然短暂，但是陈欣儿和凌仲文曾经就读于同一所小学的同一个班级。两人相处的时间只有两年，不，或许只有更短的一年而已。这就是为什么陈欣儿不明白凌仲文是那样的看重自己。完全分不清那感情到底是虚假还是真实，也无从体会小学时代的那份感情。  
　　“或许是巧合吧，在你转校过来的时候，我正好患病在床。”伯言似乎不愿意回忆一样闭着眼睛，“母亲为了照顾我，也倒下了，母亲的病比我严重得多，最后……”  
　　“是吗？对不起……”  
　　“不，不必道歉。”伯言睁开眼睛，“我还有事就先走一步了，今天谢谢你能来这里。”  
　　“喔，再见。”  
　　“再见。”  
　　“那么，我也该回去了。再见，凌仲文，伯母。”陈欣儿整理好心情，转身按来的时候原路返回。  
　　驱车来到陈欣儿过去在朱家角的家里，陈欣儿疲惫地推开卧室的门。顾不得脱外套，她直接瘫倒在床上。用手背贴在额头上，像是遮挡远处照耀过来的亮光一样。究竟是怎么回事啊？凌希文莫名其妙地遇刺先不说，为什么认识他这么久才第一次知道他还有个哥哥？不过，我们是什么时候开始疏远的呢？陈欣儿打开了天花板的显示器，“UXOS CHROME SHY07.COM”  
　　陈欣儿盯着凌仲文的主页两眼呆滞，忽然一个词语映入了她的眼帘——法利特尔。那一瞬间，陈欣儿好像是着了魔一样，“CHROME SEARCH FAIRYTALE SITE SHY07.COM”在凌仲文的博客上搜索法利特尔关键词的命令就像咒语一样脱口而出。圣母之泪，那是法利特尔世界最美的宝石。凌伯言的话语在她的脑海中回响。  
### The Intermezzo I  
　　“MINOS START”  
　　凌伯言回到自己在朱家角的家中，进入书房，打开了嵌在墙上的硕大显示器，然后用命令启动了摆放在地下室的大型终端设备。随着一阵短暂的交响乐团试音声，MINOS的控制台出现在显示器上。“MINOS LINK OBJECT 0 MEMORY 0”  
　　“嘟——”房间四角的环绕音箱一齐响起确认音效。  
　　“呀，终于来了啊，哥。”凌仲文的脸伴随着轻佻的招呼声出现在显示器的另一端。  
　　“这么快就找到外接线路了啊，似乎给你留的线索太多了。”凌伯言只是站在那里，若有所思地应付着凌仲文的话。  
　　“因为索罗的记忆里对艾克孜的恋母情结描述有点太多了！”凌仲文把脸靠向前方，几乎要从显示器里跑出来。“姝美的表现也相当不自然……”  
　　“先不提这个了，Lynn。”凌伯言打断了对方将要展开的推理攻势，“有个坏消息要告诉你。”  
　　“什么消息？”似乎察觉到了什么，被凌伯言称作Lynn的凌仲文小心翼翼地问道：“怎么没看见另一个我？”  
　　“仲文他……你的父本已经不在了……”  
　　“什么？”  
　　“我说仲文他已经死了！”凌伯言握起拳头，像是要用尽全身的力气把事情说出来一样：“上个月回国的路上，仲文他搭乘的游轮发生了爆炸，在混乱中仲文遇刺坠海……从海里救上来的时候已经没有呼吸了……”  
　　“不，这不可能！”Lynn往后退去，显示器上露出他惊恐的表情。  
　　凌伯言，是凌仲文同卵双生的孪生兄弟。因此两人有着完全相同的基因，可以说凌仲文拥有的才华，凌伯言也完全拥有。按理来说，凌伯言也可以获得和凌仲文相媲美的名声和地位。但是，理想并不总是那么的美好。  
　　凌伯言没有像凌仲文那样在传统艺术和法律这条路上发展，而是选择了在患病期间用来消遣娱乐的电子游戏。凌伯言认为比起今人遗忘的传统艺术，以电子游戏为代表的互动艺术更适合今后人们越来越慵懒的体验需求。因此凌伯言在病愈后，全身心地投入到了游戏开发的学习中。大学毕业后，由于国内游戏业界的现状并不令人满意，凌伯言选择前往作为传统电子游戏重镇的日本继续深造。  
　　而在东京工业大学攻读博士学位期间，凌伯言见到了游戏主机和游戏体验全新的发展方向——NERDLES。通过多番努力，凌伯言得知NERDLES是一种与人脑直接交换信息的VR技术，完全颠覆了人们过往对I/O设备的认识。凌伯言意识到，如果将其小型化并改进为AR技术，实现现实与虚拟的融合，那么互动艺术的舞台将会极大地扩展，NERDLES必将对未来生活产生更深远的影响。  
　　回国后，凌伯言向凌仲文说明了这件事。凌仲文则从自己的角度阐述了自己的想法：AR技术确实可以增强互动艺术的表现力，同时互动艺术也可以作为传统艺术的载体。这对新老艺术的融合是件了不起的事，但是凌仲文不认为目前国内有这方面的研究人员和从业者。毕竟国内在计算机网络游戏中浸淫太久，厂商多以软件开发为主，硬件研发实力和掌握的专利数与日本相比落后太多。如果没有得到相关的核心技术，一切从零开始恐怕追不上日本的脚步。  
　　对此凌伯言也持有同样的想法。如果从零开始，等到产品面世的时候，恐怕日本的硬件已经进入下一个世代了。所以凌伯言想：那么我们就换一个思路出发。不追求过程，只追求结果。艺术说到底就是和他人分享美的情感与意识体验。那么，艺术家如果直接把自己对作品的记忆分享给他人，不是更能传达艺术家的思想情感和意识体验吗？  
　　凌仲文也同样接受了凌伯言的想法。那一晚，兄弟二人在朱家角的家里畅谈甚欢。一边品味着祖母自己酿制的米酒，一边讨论如何利用国内现有的技术资源。最终，凌仲文依靠自己事务所的关系网接手了一份调查委托——关于调查雪崩的委托。报酬则是，多家脑部研究机构的最新研究成果和一台名为Memory Manager，简称为MEM的设备。  
　　MEM是一台搭载着MINOS的大型量子终端设备。MINOS为MINOS Is Not Operating System的递归缩写。这是通过MEM上的电磁场与人脑直接通讯，拥有包括Memory dump、Memory flash和Memory wipe在内多种命令的MEM命令集。  
　　而现在与凌伯言交谈的Lynn则是MEM的另一种用途——人脑模拟。Memory flash和Memory wipe属于不可逆操作，因此凌氏两兄弟也没有马上冒险直接写入或抹除记忆。取而代之的是，凌伯言使用MEM把凌仲文的记忆转储出来，使用MEM设定判断逻辑后模拟出凌仲文的意识。更简单的理解就是，眼下的Lynn是运行在MEM中的以凌仲文记忆为数据库的AI。  
　　“是我害死了仲文。”沉默了相当长的一段时间，凌伯言终于开口了：“如果我没有突发奇想，说什么开发另一种互动艺术的体验方式，仲文就不会去调查雪崩，也不会出国，更不会死。造成现在这样的结果，完全是我的错！”  
　　“Lynn，求求你！”凌伯言跪了下来，“Lynn，求求你，用仲文的方式责骂我吧！无论怎样难听的话都说出来吧，求你了！”  
　　“但是，”Lynn终于从打击中恢复过来，说出了令凌伯言震惊的更为悲伤的话语，“我还活着啊，哥。如果没有你把我的意识上传到MEM，我也终有一天会死的啊。但是，你看，我们现在不是还在交谈吗？你给我带来的是永生而不是死亡啊，哥。”  
　　“Lynn……仲文！”  
　　凌伯言再也抑制不住自己，眼泪如决堤的洪水一样奔涌而出。他捂着脸俯下身去，像一个孩子那样嚎啕大哭起来。  
### The Theme I  
　　“总之，请你把凌伯言的联系方式告诉我就行了！”  
　　看着女子真挚的眼神，陈宇遇到了自加入事务所以来第一次感到不可思议的事情。眼前的女子不是别人，正是前两天自己主动联系的，然后在仲文葬礼上有过一面之缘的陈欣儿。原以为和仲文早已疏远的她，能来参加葬礼已经是尽最后的礼数了。但陈宇没有想到，这个没有学过网络技术的，甚至在网络方面有些迟钝的女子，竟然破解了仲文博客的管理员密码，直接查看了仲文保存在服务器上的隐藏资料。  
　　女人的好奇心真是可怕！陈宇在心底里如此感叹。然后叹了口气，摆出认输的手势，松开了原本打算锁上的嘴，“我可以把他的联系方式给你，但是我需要知道你对事件究竟了解到了什么程度。”  
　　陈欣儿则是得意地抱着双臂，摆出“早这样不就好了”的样子，然后坐回沙发里。“凌仲文在博客的资料我都看了。凌仲文的遗愿应该是完成Project 7吧。”陈欣儿从资料中得知的是圣母之泪和法利特尔这些关键字的信息，以及Project 7的来龙去脉。当然还有一些关于雪崩的调查笔记和MEM的相关信息。  
　　Project 7是凌仲文和凌伯言联合策划的游戏项目——Love Whisper的开发代号。LOW，Love Whisper的简称，是凌仲文以索罗·希文为视角在法利特尔世界活动的记忆。根据凌伯言的构想，在时机成熟后，例如海马体神经芯片被大众和舆论广泛接受，就可以通过MEM的小型化终端把LOW的分发版覆写到体验者的海马体神经芯片中去。这样，LOW就算是完成了。  
　　虽然体验者没有实际与LOW发生互动，但是却可以体验到凌仲文在法利特尔世界游历的经历。几乎不需要花费体验者的时间，却可以获得体验者所追求的思想情感与意识体验。由于是保存在海马体神经芯片中，因此体验者如果不去主动转移，那么很快就会遗忘。这种相对安全且允许快餐存在的记忆操作正是凌伯言所追求的符合人们日益慵懒的艺术体验需求。当然，这也是让挑剔的鉴赏家们感受与艺术家进行思想交流的绝佳方式。  
　　“我想去感受凌仲文生前所有的一切，尤其是他对我到底抱着是怎样的感情。”  
　　陈欣儿如此向陈宇坦诚布公，但是这让陈宇问出了一个想当然的问题：  
　　“你不会是等他死了才爱上他吧？”  
　　陈宇口无遮拦的问话换回来的自然是陈欣儿白眼，“我可是有男朋友的。”   
　　“抱歉，和仲文相处久了都有这样的毛病，凡事都要调查一下动机。”  
　　“动机嘛，前面对凌仲文的好奇是一方面原因，另一方面的原因是……”陈欣儿说到这里忍不住脸红了起来，用手半捂着嘴说道：“另一方面我想我喜欢上法利特尔这个世界了，或者说向往更为确切一点。”  
　　“哈？”陈宇竖起的耳朵像是要掉了下来，“不是喜欢仲文本人而是喜欢他瞎编的故事，这算是哪出？”  
　　“我也觉得不可思议，”陈欣儿把手放了下来，仰起头看着天花板，继续说：“当我从凌伯言那里听到圣母之泪这个词时，我就产生了想要去了解的冲动。更不可思议的是我居然轻易地破解了凌仲文的管理员密码。在浏览资料的时候，那种像是被恶魔驱使着读下去的冲动，让我现在都还心跳不已。我想知道自己这是怎么了，为什么心里这样渴望着见到那个世界。”  
　　陈欣儿低下头看着陈宇，投以比之前更为真挚的眼神，“所以我想最好的办法就是完成LOW，完成凌仲文的遗愿，去当LOW的第一位体验者。”  
　　“是吗？”陈宇点了点头，“我感受到你的心情了，也理解了为什么仲文一直对你念念不忘。”  
　　“咦？”  
　　“仲文和我，还有伯言三个人有一次在喝酒时讨论自己理想的女性，你猜猜仲文是怎样说的？”  
　　陈欣儿摇了摇头，示意陈宇继续说下去。  
　　“高贵而优雅，”陈宇看向墙上挂着的玉兰素描，“就像玉兰那样。仲文解释说：高贵而优雅，那是看着就让人心醉的美，与之交流更是像品味美酒一样的享受，沉醉其中而浑然不觉。陈欣儿，刚才和你的交流中，我可以感受到那样的气质或气场。说实话，我也快要被你迷住了……啊，失言了。总之，你有资格加入到我们之中，谢谢。”  
　　“哪有？”陈欣儿被陈宇说得一下子手足无措，连忙拿起茶几上的杯子做出喝水的样子。  
　　“伯言和仲文的家离这里不远，如果可以的话，我送你过去吧。”  
　　“谢谢，那就麻烦你了。”  
　　“为公主效劳是骑士的义务与荣耀。”陈宇站起身，正了正衣领，走到门边替陈欣儿打开了门。
